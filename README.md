# HBG1/HBG2 Fetal Hemoglobin Regulatory Analysis

## Overview
This repository contains computational analysis identifying regulatory elements controlling fetal hemoglobin genes HBG1 and HBG2 using the Enformer deep learning model.

## Requirements
- Python 3.8+
- TensorFlow 2.x
- tensorflow-hub
- biopython
- pandas
- matplotlib

## Installation
pip install -r requirements.txt

## Key Findings
- Identified enhancer at chr11:5,308,936-5,309,374 (H3K27ac = 0.661)
- Confirmed LCR region at chr11:5,151,386-5,151,824
- Both HBG1 and HBG2 share downstream regulatory elements
